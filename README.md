# Expense-Tracker-Python

A command-line Expense Tracker built using Python with persistent JSON storage and category-wise spending visualization.

## Features
- Add expenses with category, date, and description
- Persistent storage using JSON
- Delete expenses by ID
- View all expenses
- Budget tracking
- Pie chart visualization of spending categories
## Technologies Used
- Python
- JSON
- Matplotlib
- Dataclasses
## Project Structure
```bash
├── main.py
├── expense_tracker.py
├── utils.py
├── expenses.json
├── requirements.txt
└── README.md
```
## Installation
```bash
pip install -r requirements.txt
```
